Se le proporciona un array que contiene n enteros positivos.

Su tarea es dividir el array en k subarreglos de modo que
la suma maxima en cada subarreglo sea lo mas pequeña posible.

Input
La primera linea de entrada contiene dos enteros n y k.

La siguiente linea contiene n enteros x1, x2,…, xn: el
contenido del array.

Restricciones:

- 1 <= k <= n <= 2*10^5
- 1 <= xi <= 10^9

Output
Imprima un entero: la suma maxima en una submatriz en
la division optima.

Ejemplo
Entrada
5 3
2 4 7 3 5
Salida
8
